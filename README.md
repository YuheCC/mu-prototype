# MU Agent

产品原型，包含三个主模块：

1. **Ask** — AI 对话页，类似 ChatGPT 的聊天界面
2. **Tasks & Data** — 任务与数据，含 **Tasks** 和 **Favorites** 两个子页
3. **Features & Workflows** — 功能与工作流，含固定 **Features**（如 Life Prediction、Formulate、Electrolyte Design）和用户 **Workflows**（如 Daily Data Analysis、Daily Testing Monitoring）

## 运行

```bash
npm install
npm run dev
```

在浏览器打开终端提示的地址（一般为 http://localhost:5173）即可查看。
